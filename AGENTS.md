In questo progetto ci sono sia client che server.
Il client è bun + vite, usa react router per gestire il routing ed utilizza shadcn (con tailwind) per i componenti.
Per gestire i form usa sempre react hook form e le query invece con tanstack/react-query.
Il codice deve essere semplice, pulito e mai troppo lungo oppure over-enginerizzato.

Il server invece è bun + honojs.
